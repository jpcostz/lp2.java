# Trabalhos de LP

Esta é a implementação do seguno trabalho de LP do curso de Ciências da Computação da UFF.

Representação de compras, atividade 3.

## Descrição

Em Java, realizamos com OO um código que manipula um estoque, com a implementação da adição de itens, retirada e a compra.

Este programa é a execução de uma atividade que há um cabeçalho de código definido [#Principal](https://github.com/jpcostz/lp2.java/blob/main/src/Principal.java), baseado neste código as outras classes foram definidas.
